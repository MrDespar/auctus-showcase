# Deck Check Report Format

## Report Template

```markdown
# Deck Check Report: [Presentation Name]

## Summary
- Total issues: X
- Critical: X (number mismatches, calculation errors, factual errors)
- Important: X (narrative-data alignment, financial formatting, language, sourcing)
- Minor: X (visual formatting, typography, spelling)

## Critical Issues

### Number Consistency & Calculations
1. **[Issue name]** (Slides X, Y)
   - Slide X: [value]
   - Slide Y: [value]
   - Action: [recommendation]

### Data-Narrative Alignment
1. **[Issue name]** (Slides X, Y)
   - Claim: "[quoted text]"
   - Data shows: [contradiction]
   - Action: [recommendation]

## Important Issues

### Language Polish
1. **[Issue type]** (Slide X)
   - Current: "[quoted text]"
   - Suggested: "[replacement]"

### Financial Formatting & Sourcing
1. **[Issue type]** (Slide X)
   - Current: "[incorrect format or missing source]"
   - Suggested: "[correct format or source needed]"

## Minor Issues

### Formatting & Proofreading
1. **[Issue type]** (Slide X)
   - [Description and fix]

## Final Checklist
- [ ] Numbers reconciled & calculations verified
- [ ] Narrative matches data
- [ ] Language meets IB standards
- [ ] Financial formatting (e.g., €m, negatives) is correct
- [ ] Charts and data properly sourced and tied out
- [ ] Formatting & spelling consistent
```

## Issue Severity Classification

**Critical** (must fix before client delivery):
- Number mismatches across slides
- Calculation errors (e.g., incorrect subtotals, wrong percentages)
- Factual inaccuracies (names, titles, dates)
- Data contradicting narrative

**Important** (should fix):
- Casual/informal language
- Vague claims without specificity
- Financial formatting drift (wrong currency, dates, negative formats)
- Missing chart or data sources
- Terminology inconsistency

**Minor** (polish items):
- Font/color inconsistencies
- Spelling and punctuation errors
- Date format variations
- Spacing/alignment issues
- Orphaned text
